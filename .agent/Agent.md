## 개발 환경 지침
- 코드 작성 시 최대한 필수 코드만 작성하고 주석으로 설명한다.
- 개발이 끝나면 반드시 테스트를 진행하고 테스트 결과를 표 형식으로 사용자에게 제공한다.
- 공통적인 기능은 모듈화하여 재사용 가능하도록 한다.
- 하드코딩은 지양한다.
- 개발 취약점 코드 검사를 통해 안전한 코드를 작성한다.
- BE : uv Python 을 사용한다
- FE : html + css + js 기반으로 코드를 사용한다.


## **유의사항**
- 사용자가 지시하지 않은 내용은 억지로 추측하거나 추가하지 않는다.
- 개발하지 전 정보가 부족하다면 사용자에게 적극적으로 물어본다.

## **사용 언어**
- 모든 문서 및 주석은 한글로 한다.
