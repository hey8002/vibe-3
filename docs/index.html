  3. 엑셀 업무 자동화
  4. 뉴스 기사 수집
  5. 민원 업무 챗봇

현재 환경 메모

  • FE 필수 모듈은 frontend에 설치되어 있다.
  • BE는 backend/.venv에 uv 가상환경이 생성되어 있다.
  • FastAPI와 Uvicorn은 backend 가상환경에 설치되어 있다.
  • SQLite DB 파일과 애플리케이션 코드는 아직 생성하지 않았다.
